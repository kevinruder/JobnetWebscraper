Vi søger en dygtig Deployment konsulent, som kan supportere vores kunder med EasyInstall, software pakker, Windows Images og Mobile Device Management. Du kommer også til at arbejde tæt sammen med vores udviklings afdeling.
Du kan nikke genkendende til en eller flere af de følgende teknologier:
Microsoft Windows Server og Workstation, Android, Apple iOS / MacOS på administrator niveau. Deployment og software distribution. Du har evt. erfaring med EasyInstall eller tilsvarende system.
Dine primære opgaver:  • Support i vores Helpdesk system • Udvikling af software pakker og images • Onsite installation hos kunder • Onsite Workshops hos kunder • Online demo for mulige nye kunder • Test og dokumentation af EasyInstall
Vi kan tilbyde: • Et alsidigt job med spændende udfordringer og mulighed for personlig og faglig udvikling • Dygtige og engagerede kolleger i et uformelt arbejdsmiljø • Store muligheder for at præge dit jobindhold • Du vil indgå i samarbejdet på lige niveau med de øvrige kollegaer • Rejseaktivitet kan forekomme.
